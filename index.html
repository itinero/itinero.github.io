Open source routing engine for .NET

We are Itinero and we can help you or your customers get from A to B using .NET/Mono/Xamarin on any platform!

Routing

A powerful A -> B routing engine usable as an API on the web or offline on mobile devices. Supports any vehicle profile including pedestrian, bicycle, car and truck routing.

Read More

Public Transport

Use transit schedule data together with the routing engine to calculate multimodal public transport trips. Supports combinations with any vehicle profile including pedestrians, bicycles, cars.

Read More

Optimization

Calculate large time and distance matrices and use this to optimize pickup and delivery of packages, touristic routes, trash pickup and mail delivery.

Read More

Offline on mobile

Itinero makes it possible to calculate routes for large routing networks without a connection to a routing API. This enables offline navigation and robust connection independant routing.

Optimization Engine

The routing engine provides a very fast way of calculating sets of travel distances or travel times between large collections of stops. A simple optimization engine is available to calculate solutions for a number of standard optimization problems.

Intermodal journey planner

Using transit data Itinero can be used as an intermodal journey planner. Combine any vehicle profile with transit routes and schedules to plan routes using a combination of different travel modes.
